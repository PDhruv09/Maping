🚴 Boston Bike Traffic Map

This project is an interactive geospatial visualization of BlueBikes traffic in Boston and Cambridge. It uses real-world data from March 2024 and is built using Mapbox GL JS and D3.js.

📊 Features

  • Mapbox-based background map with bike lanes overlayed (Boston & Cambridge)
  • Real-time SVG circles for every BlueBike station
  • Circle size represents total traffic; color shows traffic flow direction
  • Interactive time slider filters rides within 1-hour window
  • Legend explains color scale (arrivals vs. departures)

📁 Datasets Used

  • BlueBikes Traffic – March 2024 (CSV)
  • BlueBikes Stations (JSON)
  • Boston Bike Lanes (GeoJSON)
  • Cambridge Bike Lanes (GeoJSON)

🛠️ Tools Used

  • Mapbox GL JS
  • D3.js (v7)
  • GeoJSON + CSV + JSON data
  • GitHub Pages for hosting
